Project 01

RISC-V Vector Accelerator for ML Kernels

Architecting a decoupled vector-scalar module for silicon-efficient inference.

Abstract

Traditional CPU architectures struggle with the data-parallel nature of modern machine learning workloads. This project presents a RISC-V compatible vector accelerator implementation designed for native integration into a 5-stage pipeline. By offloading SIMD operations to a dedicated vector unit, we achieve significant throughput gains while maintaining a low silicon area footprint.

The system utilizes a decoupled interface between the scalar core and the vector accelerator. This allows the scalar unit to continue fetching and issuing commands while the vector unit processes large tensor blocks in parallel.

Signal flow & optimization

Pipeline sync

Implementation of a stall-free arbiter to manage data dependencies between scalar GPRs and vector VRFs.

Memory subsystem

Development of a strided-access load/store unit to maximize bandwidth during matrix multiplication kernels.
