Project 01 · Neuromorphic / BioSignal

NeuroSentinel Core

Real-time seizure detection engine built on Spiking Neural Networks, processing raw EEG signals through biologically-inspired spike encoding and temporal pattern recognition.

Abstract

Epileptic seizure detection demands sub-second latency with high sensitivity — a challenge for conventional deep learning models running on power-hungry GPUs. NeuroSentinel takes a different approach: it encodes raw EEG channel data into sparse spike trains and feeds them through a Spiking Neural Network that mimics biological neural dynamics. The result is a lightweight, event-driven classifier that can distinguish ictal (seizure) from inter-ictal (normal) brain states in real time, making it suitable for wearable and implantable neurodevices.

Spike encoding

Raw EEG voltage traces are converted into temporal spike patterns using rate coding. Each electrode channel maps to a neuron input — high-amplitude activity produces denser spike trains, preserving the temporal structure that traditional FFT-based approaches discard.

SNN classification

The encoded spikes propagate through layers of Leaky Integrate-and-Fire (LIF) neurons. Unlike conventional ANNs, computation only happens when spikes arrive — no spike, no energy spent. The network learns seizure-specific temporal signatures during training via surrogate gradient descent.

Why it matters

Traditional seizure detection relies on bulky models that need cloud processing. SNNs offer a path toward on-device, always-on monitoring with orders-of-magnitude lower power consumption. This project demonstrates that biologically-plausible neural computation isn't just theoretically interesting — it's practically viable for safety-critical medical applications where every millisecond of detection latency matters.
