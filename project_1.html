Project 01 · Neuromorphic / BioSignal

EEG-Based Seizure Prediction using Spiking Neural Networks

Real-time seizure detection engine that processes raw EEG signals through biologically-inspired spike encoding and Leaky Integrate-and-Fire neurons, achieving 92.35% accuracy while consuming only 151.53 µJ per inference on edge hardware.

Overview

Epileptic seizure detection demands sub-second latency with high sensitivity — a challenge for conventional deep learning models running on power-hungry GPUs. This project implements a neuromorphic approach: raw EEG channel data from 23 electrodes is converted into sparse spike trains and processed through a Spiking Neural Network that mimics biological neural dynamics. The result is a lightweight, event-driven classifier that distinguishes ictal (seizure) from inter-ictal (normal) brain states in real time, making it suitable for wearable and implantable neurodevices.

Key Implementation Details

  • EEG Filtering: Butterworth bandpass filter (0.5–40 Hz) to isolate relevant neural frequencies
  • Window Segmentation: Time-windowed analysis to identify ictal and non-ictal segments for training and validation
  • Spike Encoding: Rate coding converts EEG voltage into temporal spike patterns — high amplitude = denser spikes
  • SNN Architecture: 3-layer feedforward network with Leaky Integrate-and-Fire neurons
  • Gating Mechanism: Spike-driven LIF neurons selectively process informative windows, reducing computation
  • Training Method: Surrogate gradient backpropagation via snnTorch framework
