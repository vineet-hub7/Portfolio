Project 03

High-Speed ADC Interface for RF Telemetry

Precision analog-to-digital bridge with multi-phase interleaving logic.

Abstract

High-frequency RF capture requires precision timing and massive data bandwidth. This project implements a hybrid signal interface bridge that manages 8-way interleaved ADC cores. By implementing digital calibration logic directly in the FPGA fabric, we mitigate interleaving artifacts and noise spurs.

System flow

Hybrid signal pathway
Analog In
ADC Bridge
Digital Out

The interface supports up to 5 GSPS (Giga Samples Per Second) by distributed clock phases. A custom JESD204B-like protocol layer was implemented to deserialize the high-speed data stream into internal 256-bit buses.
