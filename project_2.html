AsyncLink PHY

A fully configurable UART transmitter module designed in Verilog — asynchronous serial communication from first principles.

Abstract

Serial communication remains the backbone of embedded systems — from sensor interfaces to debug consoles. AsyncLink PHY is a ground-up Verilog implementation of a UART transmitter with configurable baud rate, data width, and parity. The design emphasizes clean FSM architecture, precise bit-level timing, and synthesizability for both FPGA and ASIC targets.

FSM control

The transmitter operates through four states: IDLE → START → DATA → STOP. A clean Mealy-type FSM manages transitions, ensuring the start bit, data bits (LSB-first), optional parity bit, and stop bit(s) are serialized with cycle-accurate timing.

Baud rate generator

A programmable clock divider derives the baud tick from the system clock. Supporting standard rates from 9600 to 115200 baud, the divider uses a counter-based approach to produce symmetrical bit periods regardless of the input clock frequency.

Design philosophy

Building a UART from scratch isn't about reinventing what already exists — it's about deeply understanding how digital systems manage time without a shared clock. Every design decision here, from the oversampling strategy to the FSM encoding, was made with synthesizability and timing closure in mind. This is the kind of fundamental IP that underpins every SoC, and getting it right at the RTL level builds the intuition needed for more complex protocol engines.
