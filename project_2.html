Project 02 · Digital / FPGA

UART Transmitter

Hardware serial transmitter designed and implemented in Verilog with an internal baud rate generator, protocol-correct 8N1 framing, and full testbench verification in Xilinx Vivado.

Overview

A UART (Universal Asynchronous Receiver-Transmitter) transmitter is a fundamental building block in digital communication systems. This project implements a complete transmitter from first principles in Verilog, including clock-to-baud rate generation, Finite State Machine control, and protocol-compliant serial output. The design processes parallel 8-bit data input and converts it into a serial stream with start, data, and stop bits, following the 8N1 (8 bits, no parity, 1 stop bit) standard.

Implementation Details

  • Baud Rate Generator: Derives exact baud rate from system clock through divisor counter
  • FSM States (4 states):
    • IDLE — waiting for transmission request
    • START — transmits start bit (0)
    • DATA — shifts out 8 bits LSB-first
    • STOP — transmits stop bit (1)
  • Shift Register: 8-bit register with LSB-first serialization to match protocol
  • Timing Control: Baud rate clock controls state transitions and bit shifts
  • Data Format: 8N1 — 1 start + 8 data + 1 stop = 10 bits per character

Verification & Testing

Testbench Approach

A comprehensive self-checking testbench was developed to verify end-to-end functionality:

  • Generate multiple test patterns (0x00, 0xFF, 0xAA, 0x55, and random data)
  • Monitor serial output for correct framing: start bit, 8 data bits, stop bit
  • Verify bit timing against baud rate
  • Check LSB-first serialization order
  • Validate FSM state transitions

Timing Analysis

Waveform-level timing analysis was performed in Vivado to validate setup/hold margins and ensure no metastability issues in the baud rate generation logic. Timing closure achieved with positive slack on all critical paths.
