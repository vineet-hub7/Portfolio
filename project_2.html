Project 02

Neuromorphic SNN Core for Edge AI

Asynchronous event-driven hardware for ultra-low power neuromorphic compute.

Abstract

Edge devices operating on energy-harvesting sources require milliwatt-scale intelligence. This project explores the implementation of a Spiking Neural Network (SNN) core that processes data only upon asynchronous "events" or spikes, drastically reducing switching activity compared to synchronous clock-based architectures.

Architecture

Asynchronous spoke-wheel arbiter
Crossbar

The core utilizes a mesh-based interconnect of Integrate-and-Fire neurons. Weights are stored in a compressed SRAM array, utilizing the high sparsity of SNN activations to bypass redundant memory reads.

Results

Energy Efficiency
12 pJ/SOI
Latency
0.8 ms
Chip Area
0.15 mm²
